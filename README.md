# Tukvonowapi Agent
AI Agent for playing 'tukvonowapi'

## Installation Steps

Install the project
```
$ make build
```

Activate virtual environment
```
$ source ./temp/bin/activate
```

## Usage

Run the project
```
$ make run ARGS="<file_name> <B|W>"
```

Clean up / uninstall the project
```
$ make clean
```
